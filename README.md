# K-ON (在韩留学生服务与社区)

欢迎使用 K-ON 系统！这是一个专为在韩中国留学生打造的高清高保真一体化服务社区 App。本项目目前已完成**全栈改造**，对接了 **Express 后端服务** 与 **Supabase 数据库持久化存储**。

---

## 🚀 快速启动指南 (Quick Start)

### 1. 安装项目依赖
首先在项目根目录下打开终端，安装前端与后端所需的所有依赖包：
```bash
npm install
```

### 2. 配置数据库环境 (可选)
本系统采用 **双模自适应** 设计，即使不配置任何数据库凭证，也能在启动后自动进入高仿真的**本地内存数据模式**运行！

若希望接入真实的云数据库，请执行以下步骤：
1. 注册并登录 [Supabase](https://supabase.com/)，新建一个数据库项目。
2. 打开项目的 **SQL Editor**，新建一个查询，复制根目录下的 [supabase_schema.sql](supabase_schema.sql) 文件内容并粘贴，运行以一键生成所有数据表与演示数据。
3. 打开根目录下的 [.env](.env) 文件，填入你的 Supabase 项目 URL 和 anon 密钥：
   ```env
   SUPABASE_URL="https://your-project-id.supabase.co"
   SUPABASE_ANON_KEY="your-supabase-anon-key"
   ```

### 3. 运行开发服务
建议分窗口（或并排执行）启动后端 API 服务和前端界面：

*   **启动 Express 后端 API 服务器 (Port 5000)**：
    ```bash
    npm run dev:server
    ```
*   **启动 Vite 前端开发服务器 (Port 3000)**：
    ```bash
    npm run dev
    ```

启动后，在浏览器中访问前端输出的控制台地址即可开启完美的在韩留学全栈社区体验！

## 🎨 核心功能板块概览

1.  **实名自动登录**：挂载时根据本地凭据自动通过接口识别已认证学生（默认登录高保真账号“张伟”），零摩擦体验。
2.  **校园生活与社交社区**：支持按类别过滤（校园生活、学习交流、闲置交易、问答求助）、匿名/实名发布新动态，并可通过接口实时点赞、收藏及撰写评论。
3.  **电子钱包模拟体验**：内置微信/支付宝模拟快捷充值、极速韩币兑换扣款、校车电子乘车码消费扣款，所有余额与流水账单跟数据库及前端强力同步。
4.  **智能倒计时与日程提醒**：用户可自定义日历日程代办提醒，系统将自动算出距离下一个提醒事项的天数并在首页横幅实时展现，更有 HiKorea 签证到期强力倒计时提示。
5.  **生活百宝箱实用指南**：提供针对国民健康保险、外币兑换、外国人登录证办理、垃圾分类、首尔租房及交通出行的超级精美指南卡片。

## 📁 目录结构解析
*   `server.ts` - 核心 Express API 后端服务，集成 Supabase 通信与内存 Fallback 引擎。
*   `supabase_schema.sql` - 完整的 Supabase 表定义、关系索引与 Mock 初始数据导入脚本。
*   `src/App.tsx` - 精美的 React 移动端社区前端代码，全部网络请求深度对接后端 API。
*   `src/data.ts` - 前端离线 Fallback 静态演示数据集。
*   `vite.config.ts` - 配置反向代理，无缝解决 CORS 跨域请求问题。
